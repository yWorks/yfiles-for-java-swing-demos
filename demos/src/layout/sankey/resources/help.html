Description

Sankey diagrams are used for visualizing flow information. The thickness of the edges is proportional to the flow quantity. Such diagrams can be produced by means of the hierarchic layout algorithm.

The example diagram shows a voters' migration flow between different political parties over the course of three elections (each column represents an election). The flow is depicted from left to right.

Things to try

  • Toggle the checked state of the checkbox in the top toolbar.
    If the box is checked, the layout algorithm should use the existing drawing as sketch for the next layout run.
    If not checked, the layout algorithm can reduce the number of edge crossings (not considering the edge thickness).
  • Select and drag a node up or down.
    A layout calculation is triggered automatically when the node is no longer dragged (i.e. the mouse button is released).
    Note, the new node position is considered only if the checkbox in the top toolbar is checked.
