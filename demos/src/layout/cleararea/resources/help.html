Description

Shows how to interactively move graph elements around a marquee rectangle in a given graph layout so that the modifications in the graph are minimal.

Every time the marquee rectangle is dragged interactively, ClearAreaLayout. will push away the other elements, so there is a free area for the rectangle.

Things to Try

  • Drag the mouse with the right button pressed to create a marquee rectangle and watch the graph give way to it.
  • Start dragging the marquee rectangle within a group node to create more space in the group.
