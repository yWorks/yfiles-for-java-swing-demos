Description

Play with the most used layout algorithms of yFiles, including hierarchic, organic, orthogonal, tree, circular and balloon styles.

The configuration panel provides access to the most important settings of each algorithm. Typically, an algorithm allows even more options in code.

Since not every layout style works well for every type of graph, we provide several suitable sample graphs.

The interactive editing capabilities of the graph component of this demo are explained in the Simple Editor demo.

Interaction

Browse the Sample Graphs

Each sample graph is coupled with a layout algorithm well suited for the graph.

Browse the Layout Styles

Test different layout algorithms for a graph and explore the various configuration options of the algorithms.

Run a layout algorithm with the selected settings by pressing the Apply button.

Restore the default settings of an algorithm with the Reset button.
