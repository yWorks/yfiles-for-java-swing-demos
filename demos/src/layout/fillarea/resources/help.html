Description

Shows how to fill free space after deleting nodes.

When nodes are deleted interactively, the FillAreaLayout algorithm tries to fill free space with graph elements by moving nearby elements towards it, with the goal to make the existing layout more compact and not changing it too much.

Things to Try

You can choose between different strategies for assigning nodes to components whose elements should preferably not be separated:

  • Single: Each node is a separate component.
  • Connected: Components are defined by the connected components.
  • Clustering: Components are defined by edge betweenness clustering.
  • Mark some nodes in the Graph and delete them.
  • Observe how the remaining nodes are rearranged filling the space once occupied by the deleted nodes.
