Description

Customize the resizing behavior of nodes by implementing the INodeSizeConstraintProvider interface and adding instances to the node decorator of the graph.

Things to try

  • Select and resize the nodes.

Node Types

  • Blue nodes cannot shrink. Additionally, neither side can become larger than three times its initial size in each resizing operation.
  • The minimum size of a green nodes is the size of its labels. It gets never smaller than that.
  • The orange node must always encompass the black rectangle. Additionally, it has predefined minimum and maximum sizes.
