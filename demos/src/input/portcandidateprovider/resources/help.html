Description

Customize the ports at which edges connect to nodes by implementing the IPortCandidateProvider interface and adding instances to the node decorator of the graph.

Things to try

  • Initiate edge creation by dragging from an unselected node (with the left mouse button pressed). Releasing the button finishes edge creation if the mouse is on a valid target and creates a bend otherwise.

Node Types

  • Red nodes cannot have any edges. You cannot start or end an edge at them. A red port candidate highlights that these nodes are invalid targets.
  • Only edges from other green nodes can end at green nodes.
  • Blue nodes provide a set of predefined ports. Only one edge can be connected to each of these ports. Occupied ports are highlighted in red.
  • Orange nodes provide dynamic port candidates. Edges can start and end anywhere inside these nodes when the Shift key is pressed. Note that these candidates are not highlighted.
  • Edges to purple nodes can only end at ports of the same color as the source port or the source node.
  • Olive nodes accept only incoming edges and at most one such edge from any given source node.
