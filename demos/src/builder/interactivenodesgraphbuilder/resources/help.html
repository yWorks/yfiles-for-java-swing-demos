Description

Shows creating a graph with class AdjacentNodesGraphBuilder.

NodesSource defines the basic nodes of the graph. Other nodes can be added by the predecessors (successors) binding of already existing node. A node that is only defined as predecessor (successor) of another node and not by NodesSource itself is removed from the graph if it is removed from the predecessors (successors). This may result in the removal of a whole graph component if a certain predecessor (successor) connection is removed.

See the Creating a Graph from Business Data chapter of the Developers Guide for more information.

Things to Try

  • Click a node in the graph to make it the current item. Inspect its predecessors and successors.
  • Click the + button of the 'Nodes Source', 'Predecessors' and 'Successors' lists to add a new data item which creates a corresponding node.
  • Select data in 'Nodes Source' and click the - button to remove it from the list. Note that the corresponding node might not be removed if the data is still a predecessor (successor) of another item.
  • Select data in 'Predecessors' (Successors) and click the - button to remove it from the list.
  • Drag data from the graph to one of the lists.
  • Drag data from the graph into the trash to remove it.
