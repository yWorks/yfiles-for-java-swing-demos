Description

This demo shows the most important configuration options for the built-in DefaultLabelStyle class.

Things to try

  • Observe the various pre-configured label styles that are using different label shapes, different colors, fonts and other settings.
  • Take a look at labels showing the text alignment features. This comes into play when the preferred size of the label is larger than the size needed for the text.
  • The three light-blue node labels on the right show the effects of text wrapping and text clipping for labels whose size is smaller than the size needed for displaying the whole text.
