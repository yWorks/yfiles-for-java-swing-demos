Description

This demo shows label styles that render the labels independent of the zoom level.

Zoom Modes

Fit into label owner
Node labels are scaled to fit into their owner if they are bigger than their owner.
Fixed below minimum zoom
Labels stop getting smaller in view coordinates if the zoom level is less than the minimum zoom value.
Fixed above maximum zoom
Labels stop getting larger in view coordinates if the zoom level is greater than the maximum zoom value.
Zoom-invariant (Fixed when outside specified range)
Labels don't scale when the zoom level is below the minimum zoom value or above the maximum zoom value.
Default Label Style
This is the DefaultLabelStyle that comes with the yFiles library. See how its behavior compares to the demo's custom label style implementations.

Things to Try

  • Zoom in and out with the different policies to see how the labels change or keep their sizes.
  • Change the Min. Zoom value to see how it affects the Fixed below minimum zoom option.
  • Select labels and see how their hit-testing and selection changes with the zoom dependent rendering.
