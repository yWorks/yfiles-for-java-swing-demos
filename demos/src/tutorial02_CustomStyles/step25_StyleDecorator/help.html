Step 25: Style Decorator

Implement a node style decorator that enhances an existing node style by adding visual decorators. In this case the decorating node style adds the "label edges" that were formerly part of class MySimpleNodeStyle to the wrapped style's visualization.

Things to Try

  • Take a look at the source code of class MyNodeStyleDecorator.
  • Uncomment the line in SampleApplication.initializeDefaultStyles that creates an instance of RectangleNodeStyle instead of MySimpleNodeStyle and observe the outcome.
