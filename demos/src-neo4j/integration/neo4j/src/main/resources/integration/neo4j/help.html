Description

This demo loads data from a Neo4j database and displays it with yFiles for Java (Swing).

If you don't have a database running, you can install Neo4j Desktop or leave the preset to connect to a public Neo4j sample database. More public sample databases are available on the Neo4j Dataset Demo server. For servers with more than one active database, the database name must also be specified.

After establishing a connection, a small set of data from the database will be displayed.

Things to Try

  • Explore the data: Change the viewport by dragging the canvas with the mouse and change the zoom level by using the mouse wheel.
  • Hover over a node or an edge to show its properties as a tooltip.
  • Hovering over a node also highlights the adjacent edges and hovering over an edge highlights the adjacent nodes.
  • Double-click a node to center the node using a RadialLayout.

Details

This demo uses the Neo4j driver for Java to connect to a Neo4j database and query it for data.

The query results are then used by a custom GraphBuilder implementation to construct a graph from the data, while assigning different node styles to nodes with different labels.

The graph layout is achieved by using OrganicLayout with substructures enabled.
